### Creating Enhanced Modules

Modules (formerly sections) now support rich metadata and organization features.

#### Module Features

1. **Rich Descriptions**
   - Add detailed module descriptions with formatting
   - Help students understand what they'll learn

2. **Module Thumbnails**
   - Upload custom thumbnails for visual appeal
   - Recommended size: 800x450px

3. **Prerequisites**
   - Set prerequisite modules that must be completed first
   - Create structured learning paths

4. **Access Control**
   - **Free Preview**: Anyone can access
   - **Enrolled Only**: Only enrolled students
   - **Locked**: Requires prerequisites

5. **Status Management**
   - **Draft**: Work in progress
   - **Published**: Live and visible
   - **Archived**: Hidden but preserved

### Creating Text Lessons

Text lessons are perfect for written content, tutorials, and explanations.

#### How to Create

1. Navigate to your course builder
2. Click "Add Lesson" â†’ Select "Text Lesson"
3. Use the rich text editor with formatting toolbar
4. Preview your content before saving

#### Features

- **Markdown Support**: Use simple formatting syntax
- **Word Count**: Automatically calculated
- **Reading Time**: Estimated based on word count
- **Rich Formatting**: Headers, bold, italic, lists, links, images, code blocks

#### Best Practices

- Keep paragraphs short (3-5 sentences)
- Use headers to organize content
- Include images to break up text
- Add code examples when relevant
- Aim for 500-1500 words per lesson

### Creating Image Lessons

Image lessons are ideal for visual content, diagrams, and step-by-step guides.

#### How to Create

1. Click "Add Lesson" â†’ Select "Image Lesson"
2. Upload multiple images (up to 10)
3. Add titles, descriptions, and captions
4. Choose gallery layout (Grid, Carousel, or Masonry)
5. Set zoom and download permissions

#### Features

- **Multiple Images**: Up to 10 images per lesson
- **Gallery Layouts**: Grid, Carousel, or Masonry
- **Image Metadata**: Title, description, caption, alt text
- **Zoom Control**: Allow students to zoom in
- **Download Control**: Allow/prevent downloads

#### Best Practices

- Use high-quality images (1920x1080 recommended)
- Add descriptive alt text for accessibility
- Write clear captions explaining each image
- Order images logically
- Use consistent image sizes

### Enhancing Video Lessons

Video lessons now support multiple quality options, subtitles, and chapters.

#### Video Quality Options

Upload multiple quality versions:
- **1080p**: Full HD (recommended for detailed content)
- **720p**: HD (good balance)
- **480p**: SD (for slower connections)
- **360p**: Low (mobile-friendly)

#### Adding Subtitles

1. Prepare subtitle files (.srt or .vtt format)
2. Upload for each language
3. Mark default language
4. Students can toggle subtitles on/off

#### Video Chapters

Add timestamps for easy navigation:
```
00:00:00 - Introduction
00:05:30 - Main Concept
00:12:45 - Examples
00:18:20 - Summary
```

### Adding Lesson Resources

Attach supplementary materials to any lesson.

#### Resource Types

- **PDF Documents**: Worksheets, guides, references
- **Documents**: Word, Excel, PowerPoint files
- **Images**: Diagrams, infographics
- **Links**: External websites, videos
- **Code Files**: Sample code, templates
- **Other**: Any other file type

#### How to Add Resources

1. Open lesson editor
2. Scroll to "Lesson Resources" section
3. Click "Add Resource"
4. Select type, upload file or enter URL
5. Add description
6. Set download permissions

#### Best Practices

- Organize resources by type
- Write clear descriptions
- Keep file sizes reasonable (<10MB)
- Use descriptive filenames
- Only allow downloads when appropriate

### Setting Completion Requirements

Control how students complete lessons.

#### Completion Types

1. **Manual**: Student marks as complete
2. **Auto Video 80%**: Complete when 80% watched
3. **Auto Document**: Complete when document viewed
4. **Quiz Pass**: Must pass associated quiz
5. **Assignment Submit**: Must submit assignment

#### Choosing the Right Type

- **Text Lessons**: Auto (scroll to bottom) or Manual
- **Video Lessons**: Auto Video 80%
- **Image Lessons**: Auto (all images viewed)
- **Quiz Lessons**: Quiz Pass
- **Assignment Lessons**: Assignment Submit

## ðŸ‘¨â€ðŸŽ“ For Students

### Viewing Text Lessons

#### Features

- **Reading Progress**: Track how much you've read
- **Word Count**: See lesson length
- **Reading Time**: Estimated time to complete
- **Auto-Complete**: Automatically marks complete when you scroll to bottom

#### Tips

- Read at your own pace
- Use the progress bar to track your position
- Scroll to the bottom to auto-complete
- Or manually mark as complete when ready

### Viewing Image Lessons

#### Features

- **Gallery Views**: Switch between grid and list
- **Lightbox**: Click any image for full-screen view
- **Zoom**: Zoom in/out on images (when allowed)
- **Navigation**: Use arrows to browse images
- **Download**: Download images (when allowed)
- **Progress Tracking**: See which images you've viewed

#### Tips

- Click images to view full-size
- Use keyboard arrows for navigation
- Zoom in to see details
- View all images to auto-complete

### Watching Enhanced Videos

#### Features

- **Quality Selector**: Choose video quality based on connection
- **Subtitles**: Toggle subtitles in multiple languages
- **Chapters**: Jump to specific sections
- **Progress Tracking**: See how much you've watched
- **Auto-Complete**: Marks complete at 95% watched

#### Tips

- Start with auto quality, adjust if needed
- Enable subtitles for better comprehension
- Use chapters to review specific topics
- Watch at least 95% to auto-complete

### Accessing Lesson Resources

#### Features

- **Grouped by Type**: Resources organized by category
- **View Online**: Preview resources in browser
- **Download**: Download for offline access (when allowed)
- **File Info**: See file size and type

#### Tips

- Download resources before class
- Check file sizes on mobile data
- View online to save storage
- Organize downloads in folders

### Tracking Your Progress

#### Progress Indicators

- **Module Progress**: See completion percentage per module
- **Lesson Progress**: Track individual lesson progress
- **Overall Progress**: View course completion status

#### Auto-Completion

Lessons auto-complete when:
- Text: Scrolled to 95% or bottom
- Images: All images viewed
- Videos: Watched 95%+
- Documents: Viewed completely

#### Manual Completion

Some lessons require manual completion:
- Click "Mark as Complete" button
- Only available after viewing content
- Cannot be undone (but can be reset by teacher)

### Common Issues

#### "Video won't play"
- Check internet connection
- Try lower quality setting
- Clear browser cache
- Try different browser

#### "Images not loading"
- Refresh the page
- Check internet connection
- Disable browser extensions
- Try incognito mode

#### "Can't download resource"
- Check if downloads are allowed
- Verify you're enrolled
- Check browser download settings
- Try different browser

#### "Progress not saving"
- Ensure you're logged in
- Check internet connection
- Complete required percentage
- Wait a few seconds before leaving
