# Video Tutorial Script: Admin Course Creation Workflow

**Duration**: 8-10 minutes  
**Target Audience**: School Administrators  
**Difficulty**: Beginner

---

## Video Outline

1. Introduction (0:00 - 0:30)
2. Creating a New Course (0:30 - 3:00)
3. Assigning Teachers (3:00 - 5:30)
4. Managing Permissions (5:30 - 7:30)
5. Monitoring and Best Practices (7:30 - 9:00)
6. Conclusion (9:00 - 10:00)

---

## Script

### Introduction (0:00 - 0:30)

**[SCREEN: St. Haroon Online School Dashboard]**

**Narrator:**
"Welcome to St. Haroon Online School! In this tutorial, you'll learn how to create courses and assign teachers as an administrator. By the end of this video, you'll be able to set up a complete course with properly configured teacher assignments."

**[TEXT OVERLAY: "What You'll Learn"]**
- Create a new course
- Assign teachers with specific permissions
- Manage teacher access
- Monitor course assignments

---

### Creating a New Course (0:30 - 3:00)

**[SCREEN: Admin Dashboard]**

**Narrator:**
"Let's start by creating a new course. From your admin dashboard, click on 'Courses' in the main navigation menu."

**[CLICK: Courses menu]**

**Narrator:**
"Now, click the 'Create New Course' button in the top right corner."

**[CLICK: Create New Course button]**
**[SCREEN: Course Creation Form]**

**Narrator:**
"You'll see the course creation form. Let's fill in the required information."

**[TYPE: Course Title]**
"First, enter a clear, descriptive course title. For this example, we'll create 'Introduction to Biology - Grade 10'."

**[TYPE: Description]**
"Next, provide a comprehensive course description. This helps teachers and students understand what the course covers."

**[SELECT: Subject dropdown]**
"Select the appropriate subject from the dropdown. We'll choose 'Science'."

**[SELECT: Grade Level]**
"Choose the grade level. For this course, we'll select 'Intermediate'."

**[SCROLL: Additional settings]**
"You can also set the enrollment capacity, course duration, and initial status. For now, we'll leave the status as 'Draft' until we're ready to publish."

**[CLICK: Create Course button]**

**Narrator:**
"Click 'Create Course' to save. Great! Your course has been created successfully."

**[SCREEN: Course Detail Page]**

**Narrator:**
"You're now on the course detail page where you can manage all aspects of this course."

---

### Assigning Teachers (3:00 - 5:30)

**[SCREEN: Course Detail Page]**

**Narrator:**
"Now let's assign teachers to this course. Click on the 'Assign Teachers' tab."

**[CLICK: Assign Teachers tab]**
**[SCREEN: Teacher Assignment Interface]**

**Narrator:**
"Click 'Add Teacher Assignment' to begin."

**[CLICK: Add Teacher Assignment button]**
**[SCREEN: Teacher Selection Modal]**

**Narrator:**
"Search for the teacher you want to assign. You can search by name or email. Let's search for 'Dr. Sarah Johnson'."

**[TYPE: Search field - "Dr. Sarah Johnson"]**
**[SELECT: Teacher from dropdown]**

**Narrator:**
"Select the teacher from the results. Now we need to configure their permissions."

**[SCREEN: Permission Configuration]**

**Narrator:**
"There are four permission options to consider:"

**[HIGHLIGHT: Can Manage Content checkbox]**
"First, 'Can Manage Content' - This allows the teacher to create, edit, and delete course materials. We'll enable this for Dr. Johnson since she's the primary instructor."

**[CHECK: Can Manage Content]**

**[HIGHLIGHT: Can Grade checkbox]**
"Second, 'Can Grade' - This permits the teacher to grade assignments and manage the gradebook. We'll enable this as well."

**[CHECK: Can Grade]**

**[HIGHLIGHT: Can Communicate checkbox]**
"Third, 'Can Communicate' - This allows messaging students and posting announcements. Essential for any teacher, so we'll enable it."

**[CHECK: Can Communicate]**

**[HIGHLIGHT: Is Primary Teacher checkbox]**
"Finally, 'Is Primary Teacher' - This designates the main instructor for the course. Since Dr. Johnson is leading this course, we'll check this box."

**[CHECK: Is Primary Teacher]**

**Narrator:**
"Review the selected permissions to ensure they're correct."

**[CLICK: Assign Teacher button]**

**Narrator:**
"Click 'Assign Teacher' to complete the assignment. Dr. Johnson will receive an email notification immediately."

**[SCREEN: Success message]**

**Narrator:**
"Perfect! The teacher has been assigned successfully."

---

### Managing Permissions (5:30 - 7:30)

**[SCREEN: Assigned Teachers List]**

**Narrator:**
"Let's add an assistant teacher with limited permissions. Click 'Add Teacher Assignment' again."

**[CLICK: Add Teacher Assignment]**
**[SEARCH: "Mr. David Chen"]**
**[SELECT: Teacher]**

**Narrator:**
"For our assistant teacher, Mr. Chen, we'll configure different permissions. He'll help with grading but won't manage content."

**[UNCHECK: Can Manage Content]**
**[CHECK: Can Grade]**
**[CHECK: Can Communicate]**
**[UNCHECK: Is Primary Teacher]**

**Narrator:**
"Notice that 'Can Manage Content' and 'Is Primary Teacher' are unchecked. This gives Mr. Chen a supporting role focused on grading and communication."

**[CLICK: Assign Teacher]**

**Narrator:**
"Now we have two teachers assigned with different permission levels."

**[SCREEN: Assigned Teachers List showing both teachers]**

**Narrator:**
"You can see both teachers listed with their permission badges. The badges make it easy to see each teacher's capabilities at a glance."

**[HIGHLIGHT: Permission badges]**

**Narrator:**
"If you need to update permissions later, simply click the edit icon next to any teacher."

**[CLICK: Edit icon]**
**[SCREEN: Edit Permissions Modal]**

**Narrator:**
"Modify the permissions as needed and click 'Update Permissions'. The teacher will be notified of any changes."

**[CLICK: Cancel to return]**

---

### Monitoring and Best Practices (7:30 - 9:00)

**[SCREEN: Navigate to Teacher Assignments Overview]**

**Narrator:**
"To monitor all teacher assignments across your school, navigate to 'Courses' then 'Teacher Assignments'."

**[CLICK: Courses > Teacher Assignments]**
**[SCREEN: Assignment Overview Dashboard]**

**Narrator:**
"This dashboard shows all course assignments with useful metrics and filters."

**[HIGHLIGHT: Metrics section]**
"You can see total courses created, total teacher assignments, and permission distribution."

**[HIGHLIGHT: Filters]**
"Use filters to find specific assignments by course, teacher, or permission type."

**[SCREEN: Audit Logs]**

**Narrator:**
"For detailed history, check the Audit Logs under Admin menu. This shows every permission change with timestamps and details."

**[SCREEN: Best Practices slide]**

**Narrator:**
"Here are some best practices to remember:"

**[TEXT OVERLAY: Best Practices]**
- Always assign at least one primary teacher per course
- Grant minimum necessary permissions
- Communicate with teachers about their assignments
- Review assignments regularly
- Use the audit log to track changes

### Conclusion (9:00 - 10:00)

**[SCREEN: Summary slide]**

**Narrator:**

**[TEXT OVERLAY: Summary]**

## Production Notes

### Equipment
- Screen recording software (Camtasia, ScreenFlow, or OBS)
- High-quality microphone
- Video editing software

### Recording Tips
- Use a clean test environment
- Prepare sample data beforehand
- Record in 1080p or higher
- Use smooth mouse movements
- Pause between sections for editing

### Post-Production
- Add intro/outro animations
- Include background music (subtle)
- Add text overlays for key points
- Include captions/subtitles
- Export in multiple formats (MP4, WebM)

### Accessibility
- Include closed captions
- Provide transcript
- Use high contrast for text overlays
- Ensure clear audio quality
- Add audio descriptions if needed
