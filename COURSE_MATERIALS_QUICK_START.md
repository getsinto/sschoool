## ğŸ“ Creating a Worksheet

### Step 1: Navigate to Worksheets
```
Course Builder â†’ Worksheets Manager â†’ Create Worksheet
```

### Step 2: Fill in Basic Information
- **Title**: Give your worksheet a clear name
- **Description**: Brief overview of what it covers
- **Instructions**: Detailed steps for students
- **Difficulty**: Easy, Medium, or Hard
- **Estimated Time**: How long it should take

### Step 3: Upload Files
- **Worksheet File**: PDF, DOC, DOCX, PNG, or JPG (max 50MB)
- **Answer Key** (optional): Hidden from students

### Step 4: Configure Settings
- **Require Submission**: Toggle if students must submit completed work
- **Max Grade**: Set points (default: 100)
- **Allow Download**: Let students download the file
- **Allow Print**: Let students print the worksheet

### Step 5: Add Tags (Optional)
- Add keywords for easy searching
- Example: "fractions", "homework", "chapter-3"

### Step 6: Save
Click "Create Worksheet" - Done! âœ…

### Step 1: Navigate to Resources
```
Course Builder â†’ Resources Library â†’ Add Resource
```

### Step 2: Choose Resource Type
- **External Link**: Websites, articles, YouTube videos
- **File Upload**: PDFs, documents, any file type
- **Video**: Video resources
- **Document**: Document resources
- **Reference**: Study materials

- **Title**: Resource name

### Step 4: Add Content
- **For Links**: Paste the URL
- **For Files**: Upload file (max 100MB)

### Step 5: Configure Access
- **Requires Enrollment**: Only enrolled students can access

## âš ï¸ Common Issues

### File Upload Fails
- **Check file size**: Max 50MB for worksheets, 100MB for resources
- **Check file type**: PDF, DOC, DOCX, PNG, JPG supported
- **Try again**: Refresh and retry upload

### Can't Find Worksheet
- **Check search**: Clear search box
- **Check filters**: Reset all filters
- **Check module**: Make sure you're in the right module

### Submission Not Showing
- **Refresh page**: Click refresh button
- **Check course**: Verify correct course selected
- **Wait**: Student may not have submitted yet
